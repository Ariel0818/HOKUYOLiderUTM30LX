# docs

Setup and troubleshooting guides for hardware and SLAM software.

|----------|-------------|
| `Lidar_instruction.pages` | Hokuyo UTM-30LX setup guide (Chinese) |
| `Lidar_instruction_English.pages` | Hokuyo UTM-30LX setup guide (English) — LiDAR connection, `urg_node2` launch, ROS 2 topic verification |

| `rtabmap_install_failed.pages` | Troubleshooting notes for Rtabmap installation failures on ROS 2 Humble |
| `2.2Canopydata.pages` | Notes on canopy data structure and file organization |
